# FinalYearProject

> 我的畢業專題為開發一個老人照護的穿戴式裝置及裝置的連動APP。
> 此作品的目標族群為「家裡有老人需要照顧」的人，所以我們讓使用者可以即時
> 知道家中老人的身體狀況及地點位置讓意外發生時能知道他在哪裡。

 

## Andorid App
### 功能
- 接收Arduino端的心跳資訊
- 將資訊傳遞至firebase資料庫

 

##  Arduino

### 功能
- 測量使用者心跳數值
- 利用藍芽模組(HC-06)傳至Android App

 
 
 ## Firebase
 
 ### 功能
 - 儲存使用者之帳號、相關量測數據。
